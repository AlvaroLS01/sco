# Addon de SelfCheckout

Este proyecto genera un jar que, añadido a una carpeta de instalación del POS de comerzzia (en la carpeta lib), añade las **personalizaciones** necesarias para tener una aplicación de SelfCheckout.

### Instrucciones

1. Este SelfCheckout solo tiene activo el pago con tarjeta, relacionado con el medio de pago 0010. Para cambiar este código de medio de pago, habría que cambiar en el skin el siguiente panel: %nombre_skin%\com\comerzzia\pos\selfcheckout\gui\pagos\selfcheckoutpagos_sc_panel.xml

```xml
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<Panel>
	<LineaBotones>
		<Boton>
			<texto>TARJETA</texto>
			<clave>seleccionarMedioPagoSelfCheckout</clave>
			<tipo>METODO</tipo>
			<rutaImagen>importes/creditcard.png</rutaImagen>
			<parametros>
				<entry>
					<key>codMedioPago</key>
					<value>0010</value>
				</entry>
			</parametros>
		</Boton>
	</LineaBotones>
</Panel>
```

2. Hará falta dar de alta un nuevo usuario con el código de usuario SELF_CHECKOUT y la contraseña selfcheckout. Este usuario debe tener el perfil Usuarios.
